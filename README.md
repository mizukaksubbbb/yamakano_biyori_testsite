# yamakano-biyori-test

Static site for GitHub Pages.

## Deploy
Enable GitHub Pages:
- Settings → Pages → Build and deployment → Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

Then visit: https://<YOUR_GITHUB_USERNAME>.github.io/yamakano-biyori-test/
